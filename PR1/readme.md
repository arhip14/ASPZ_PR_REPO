Було встановлено систему Ubuntu встановлено на віртулаьну машину UTM.
Налаштовано основні параметри віртуальної машини, встановлено потрібні компоненти та оновлено систему. Також було підготовлено середовище 
для роботи з мовою програмування C.

Завдання 1.10 виконано повністю. Написано програму мовою ANSI C з використанням перелічуваних типів (enum),
яка визначає та виводить завтрашню дату на основі введеної користувачем.
Програма підтримує роботу з григоріанським та юліанським календарями та враховує високосні роки.
Роботу програми перевірено в терміналі Ubuntu та в середовищі Visual Studio Code.
