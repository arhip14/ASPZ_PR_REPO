Практична робота №6

Завдання 10

Реалізувати мінімальну оболонку з підтримкою fork/exec, пайпів, перенаправлення вводу-виводу та коректної обробки сигналів.

Опис

Програма реалізує мінімальну оболонку на C, яка підтримує запуск команд через fork та exec, перенаправлення вводу/виводу (< та >), 
пайпи (|) та обробку сигналу Ctrl+C. Користувач може виконувати прості команди, комбінувати їх через пайпи та редіректи, а також виходити з оболонки натисканням Ctrl+D.

Ідея реалізації

Оболонка читає командний рядок користувача, розбиває його на аргументи, перевіряє на наявність пайпів або перенаправлень.
Для виконання команд створюється дочірній процес через fork(), а команда виконується за допомогою execvp(). 
При наявності пайпів відбувається перенаправлення стандартного вводу/виводу між процесами за допомогою pipe().
Перенаправлення вводу/виводу (<, >) реалізоване через dup2(). Сигнал Ctrl+C обробляється у батьківському процесі для запобігання завершенню оболонки, а дочірні процеси отримують стандартну поведінку сигналу.
